# ssisyg.github.io

Personal website for Junlong Ning, built as a dependency-free static page for GitHub Pages.

## Files

- `index.html` — complete responsive website
- `assets/Junlong_Ning_Resume.pdf` — downloadable resume

## Publish

Upload the files to the root of the `ssisyg/ssisyg.github.io` repository on the `main` branch. GitHub Pages will serve the site at <https://ssisyg.github.io/>.
